Launching:
	- The Avian can be launched by clicking -> dragging -> releasing.
	- While dragged, an arrow sprite will be displayed to show the estimated flight direction of the avian and its force

Background Art:
	- Created on Inkscape
	- Ground: Brown rectangle with an organic image filter
	- Grass: Custom green area with a film grain filter
	- Clouds: Custom drawn white areas with blur filter
		- In the game, clouds also have a script that moves them attached
		
Version control:
	- Git used
	- Public repository in Github: https://github.com/zanagi/SlightlyAnnoyedAvians
